# ERPmini Mobile

Projeto React pronto para rodar como site mobile.

## Como testar no computador

1. Instale o Node.js.
2. Abra esta pasta no VS Code.
3. No terminal, rode:

```bash
npm install
npm run dev
```

4. Acesse o link que aparecer no terminal.

## Como publicar grátis na Vercel

1. Crie conta em https://vercel.com
2. Envie esta pasta para um repositório GitHub ou importe o projeto direto.
3. Framework: Vite.
4. Build command: `npm run build`
5. Output directory: `dist`

## Observação

Os dados ficam salvos no navegador/celular usando localStorage. Se limpar os dados do navegador, os produtos e vendas podem ser apagados.
